# 🚀 Skill Stack — Next-Gen AI Learning Platform

An **interactive, AI-powered multilingual learning platform** that combines **gamification, real-time AI assistance, and immersive experiences** to redefine how students learn.

---

## 🌟 Overview

Skill Stack is not just another LMS — it is a **smart, adaptive, and engaging ecosystem** designed for modern learners.

It integrates:

* 🤖 AI-driven personalization
* 🎮 Gamified learning paths
* 🌍 Multilingual accessibility
* 🧠 Emotion-aware learning
* 🧪 Virtual labs & simulations

---

## 🧠 Key Features

### 🤖 AI & Personalization

* **Emotion-Responsive Learning**

  * Detects user emotions via webcam (face-api.js)
  * Adjusts learning pace dynamically

* **Offline AI Assistant**

  * Works without internet using local logic / WebLLM
  * Answers basic queries instantly

* **AI Microlearning Generator**

  * Converts long lectures into bite-sized lessons
  * Generates flashcard-style modules

---

### 🎮 Gamified Experience

* **Skill-Based Quest Map**

  * RPG-style progression system
  * Level up based on skill mastery

* **Live Trivia Arena**

  * Real-time quiz battles
  * Competitive scoring system

* **XP, Coins & Streak System**

  * Encourages consistency and engagement

---

### 🧪 Immersive Learning

* **Virtual Hands-on Labs**

  * Safe simulation-based experiments
  * Canvas-based interactive environment

---

### 🤝 Social & Collaborative

* **AI-Matched Study Groups**

  * Pairs users based on complementary skills

* **Social Annotation**

  * Add notes directly to learning content

* **User-Generated Courses**

  * Create and publish courses using AI assistance

---

### ♿ Accessibility & Inclusivity

* **Adaptive UI**

  * High contrast mode
  * Large text support

* **Multilingual Platform**

  * Supports:

    * 🇬🇧 English
    * 🇮🇳 Hindi

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/skill-stack.git
cd skill-stack
```

---

### 2️⃣ Run Locally

Just open:

```bash
index.html
```

OR use Live Server (recommended)

---

### 3️⃣ (Optional) Setup Face Detection Models

Download models from:
https://github.com/justadudewhohacks/face-api.js-models

Place inside:

```
/models
```

---

### 4️⃣ (Optional) Enable AI Chat API

Replace API key in:

```javascript
Authorization: "Bearer YOUR_API_KEY"
```
